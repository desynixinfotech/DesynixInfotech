# Desynix Infotech - Startup Website

A modern, high-performance website for Desynix Infotech, built with React, TailwindCSS, Framer Motion, and Node.js.

## Tech Stack

### Frontend
- **ReactJS** (Vite)
- **TailwindCSS** (Custom Design System)
- **Framer Motion** (Animations)
- **React Router DOM**
- **Lucide React** (Icons)

### Backend
- **Node.js**
- **Express.js**
- **REST API** (Mock Persistence)

## Setup Instructions

### 1. Backend Setup
Open a terminal and run:
```bash
cd server
npm install
node index.js
```
The server will start on port `5000`.

### 2. Frontend Setup
Open a **new terminal** and run:
```bash
cd frontend
npm install
npm run dev
```
The frontend will start on port `5173`.

## Features
- **Modern UI/UX**: Dark mode aesthetic with gradients and animations.
- **Dynamic Content**: Services, Technologies, and Portfolio pages.
- **Contact Form**: Functional form with backend API handling and local storage persistence.
- **Testimonials**: Interactive carousel with local storage persistence.
- **Responsive Design**: Mobile-first approach.

## Structure
- `frontend/`: React application source code.
- `server/`: Express backend API source code.
